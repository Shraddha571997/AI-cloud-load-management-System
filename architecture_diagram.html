Data Flow & Key Connections
Presentation Layer → Application Layer React Frontend sends HTTP requests (Axios) to Flask REST API via JSON.
Application Layer → Data Layer Flask API reads/writes to MongoDB via PyMongo for users, predictions, logs & metrics.
ML Module ⟷ ML Model File ML Prediction Module loads load_model.pkl at runtime; Admin triggers retraining & overwrites the file.
Scheduler Service → ML Prediction Module APScheduler fires cron jobs to invoke ML Prediction, triggering Scaling Engine decisions automatically.
Alert System → Email (SMTP) Flask-Mail dispatches email alerts when CPU/Memory thresholds are breached or anomalies are detected.
Scaling Engine → Decision Output Outputs one of: Maintain / Scale Up / Scale Down / Aggressive Scale based on ML prediction score.
